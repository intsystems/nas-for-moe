# Toy Experiment — общий контекст

## Суть исследования

Поиск архитектуры **Mixture of Experts (MoE)** с использованием **суррогатной функции**.

Суррогатная функция `u(alpha, R)` предсказывает валидационную точность архитектуры `alpha` при обучении на данных, определяемых вектором `R`.

## Ключевые сущности

| Обозначение | Описание |
|---|---|
| `alpha` / `alpha_k` | Архитектура (эксперта) — дискретная конфигурация |
| `M` | Число кластеров данных |
| `K` | Число экспертов |
| `b ∈ {0,1}^M` | Бинарный вектор подмножества кластеров (какие кластеры включены в обучение) |
| `R_k` | Вектор принадлежности кластеров эксперту `k` (столбец `r[:, k]`) |
| `r[m, k]` | Вероятность назначения кластера `m` эксперту `k`; строка `r[m, :]` — распределение по экспертам |
| `u(alpha, R)` | Суррогатная функция — предсказание качества архитектуры на заданном подмножестве данных |

## Особенности суррогатной функции

- Обучена **только на дискретных бинарных векторах**.
- Для недифференцируемых методов: сэмплировать бинарные вектора из вероятностей.
- Для дифференцируемых методов: использовать **Gumbel-Softmax / straight-through relaxation**.

## Неопределённость (MC Dropout)

Для оценки неопределённости предсказания суррогата:
- Включить dropout на инференсе.
- Сделать несколько стохастических прогонов.
- Посчитать среднее `mu` и стандартное отклонение `sigma`.

## Кластеризация

- Обучающие данные имеют кластерную разметку длины `N_train`.
- Валидационные точки назначаются кластеру по **ближайшему центру**.

## Код (toy_experiment/)

### Pipeline (в порядке запуска)

| Шаг | Файл | Назначение |
|---|---|---|
| 1 | `toy_generate_dataset.py` | Генерация данных: 1 linear + 1 ring cloud (balanced) |
| 1 | `toy_generate_dataset_multi.py` | Генерация данных: N linear + N ring clouds на сетке |
| 1 | `toy_generate_dataset_imbalanced.py` | Генерация данных: imbalanced (200 linear + 1800 ring) |
| 2 | `toy_clustering.py` | KMeans кластеризация + train/val split |
| 3 | `collect_dataset.py` | Сбор датасета суррогата (active learning: обучение моделей, MC Dropout UCB) |
| 4 | `retrain_surrogate.py` | Переобучение суррогата на всём датасете с нуля (grid search гиперпараметров) |
| 5 | `optimize_sampling.py` | Оптимизация MoE: метод 1 — sampling search |
| 5 | `optimize_gradient.py` | Оптимизация MoE: метод 2 — градиенты по r + Gumbel-Softmax |
| 5 | `optimize_em.py` | Оптимизация MoE: метод 3 — EM-алгоритм |
