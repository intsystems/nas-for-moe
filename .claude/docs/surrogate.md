# Суррогатная функция

## Назначение

Суррогатная функция `u(alpha, R)` предсказывает валидационную точность архитектуры `alpha` при обучении на данных, определяемых вектором `R`.

Используется вместо полного обучения модели — позволяет быстро оценить качество архитектуры без затратного train-цикла.

## Вход / Выход

| | Описание |
|---|---|
| **Вход** | Архитектура `alpha` (encoding) + бинарный вектор `b ∈ {0,1}^M` (подмножество кластеров) |
| **Выход** | Предсказание валидационной точности (скаляр) |

## Особенности

- Обучена **только на дискретных бинарных векторах** `b`.
- Для **недифференцируемых** методов оптимизации: сэмплировать бинарные вектора из вероятностей `r`.
- Для **дифференцируемых** методов: использовать **Gumbel-Softmax** или **straight-through relaxation** для прохождения градиентов через дискретизацию.

## Оценка неопределённости (MC Dropout)

Для получения не только предсказания, но и оценки уверенности:

1. Включить dropout на инференсе (`model.train()` или явно включить dropout-слои).
2. Сделать `T` стохастических прогонов (forward pass).
3. Посчитать:
   - Среднее: `mu = (1/T) * Σ u_t(alpha, R)`
   - Стандартное отклонение: `sigma = std(u_1, ..., u_T)`

Используется в active learning для выбора новых кандидатов по критерию `mu + sigma`.

## Обучение суррогата (Active Learning)

1. Сэмплировать начальные пары `(b, alpha)`, обучить модели, измерить accuracy.
2. Собрать датасет: `(architecture_encoding, cluster_subset_vector_b, validation_score)`.
3. Обучить суррогатную функцию на этом датасете.
4. Сэмплировать новых кандидатов по критерию `mu + sigma`.
5. Обучить выбранных кандидатов, добавить результаты в датасет.
6. Повторить.
