Описание алгоритма
На итерации t:

E-шаг: q^{(t)} = argmax_{q ∈ Q} F_{u^{(t)}}(q, θ^{(t)}), то есть:
q_{mk}^{(t)} = r_{mk}^{(t)} · u^{(t)}(α_k^{(t)}, R_k^{(t)}) / Σ_j r_{mj}^{(t)} · u^{(t)}(α_j^{(t)}, R_j^{(t)})

M-шаг (GEM): найти θ^{(t+1)} такое, что:
F_{u^{(t)}}(q^{(t)}, θ^{(t+1)}) ≥ F_{u^{(t)}}(q^{(t)}, θ^{(t)})

S-шаг: обновить суррогат u^{(t)} → u^{(t+1)} (добавить точки, переобучить).
